# Baştürk Holding Web Sitesi

Modern, profesyonel ve premium bir holding web sitesi.

## Özellikler

### Tasarım
- ✨ Modern ve minimal tasarım
- 🎨 Dark/Light mode desteği
- 📱 Tam responsive (mobil, tablet, desktop)
- 🎭 Özel cursor efektleri
- 🌊 Parallax scroll animasyonları
- ✨ Smooth scroll ve fade-in animasyonlar

### Fonksiyonellik
- 🌍 Çift dil desteği (TR/EN)
- 📧 İletişim formu
- 📰 Newsletter abonelik sistemi
- 🍪 Cookie consent banner
- 🔝 Back to top butonu
- 🎯 Şirket filtreleme sistemi
- 📊 Animasyonlu istatistikler
- ⏱️ Timeline (tarihçe) bölümü

### Bölümler
1. Hero - Etkileyici giriş bölümü
2. Hakkımızda - Holding tanıtımı ve istatistikler
3. Şirketlerimiz - 6 şirket detaylı kartlar
4. Tarihçe - Timeline görünümü
5. Haberler - Son gelişmeler
6. İletişim - Form ve bilgiler
7. Newsletter - Bülten kayıt

## Kullanım

Web sitesini görüntülemek için `index.html` dosyasını tarayıcınızda açın.

## Teknolojiler

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter, Playfair Display)
